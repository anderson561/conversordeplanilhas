# Frontend Component Spec - Material 3

## UI/UX Design System
Baseado nos princípios do Material Design 3, focando em acessibilidade e feedback visual premium.

## Componentes Críticos
1. **Upload Zone**: Drag & Drop com feedback háptico (visual) e estados `hover`, `active` e `error`.
2. **Progress Indicator**: Barra de progresso linear para indicar o status da conversão (0-6s).
3. **Results Card**: Card com elevação (Elevation level 2) para exibir o link de download do XML.

## Heurísticas de UX
- **Reconhecimento em vez de recall**: Mantenha os campos de mapeamento visíveis durante todo o processo.
- **Ajuda e Documentação**: Tooltips explicativos em cada coluna de mapeamento técnico.

> [!NOTE]
> Perfeição de pixel é o mínimo. A experiência de 6 segundos deve ser fluida e sem interrupções.
